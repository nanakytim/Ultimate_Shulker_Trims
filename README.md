# CONFIGURATION

### This Mod is highly customisable to your taste:

- **DEFINE YOUR OWN LOCATIONS:** Add a location by drawing its X / Y / Z coordinates and radii. Shape the box where music will start playing!
- **DIFFERENT SONGS:** Add a different song to different locations.
- **VOLUME:** Set the volume of each unique song to your taste. **There is also an Underwater Pitch modifier to add variation, that you can switch off.**
- **FADE:** Add Fade in/out to the Location music!

**_NOTE: By default, Location music does not override other music. You can toggle this to override and stop other songs when a Location triggers music._**

### COMPATIBLE WITH OTHER MUSIC MODS:

Since the Location Music by default only plays in silence, it is highly compatible with other music mods and vanilla ambient music :)

This Mod provides default music. You can replace those songs with your own by overwriting the songs with a Resource Pack! The path for each file is as follows: assets\ultimate_location_music\sounds\
- **location_1.ogg**
- **location_2.ogg**
- **etc.**
**_Note: up to 32 songs will be automatically detected!_**

It's fully client-side, no dependencies. Have fun!
